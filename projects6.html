NBC News Elections Web Apps

The goal of this project was to redesign NBC's best in class internal web apps that drive award winning Elections Coverage. I worked in the Elections Data Team collaborating with Front-end & Back-end Developers using the SCRUM methodology.

My Role

UI/UX Designer

Interaction Designer

Duration

18 months

Tools

Figma

Storybook

Execution

1. Redesigned NBC's internal web applications while ensuring a consistent theme across all of them.

2. Contributed to the development of new features within the primary elections application.

3. Included data visualization with the aim of reducing the strain of going through large amounts of data.

4. Created a design system and integrated reusable components into Storybook for our front-end developers.

5. Created high fidelity prototypes for demos and usability testing on our target users.
